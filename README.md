## 🤖 Part 1 — Telegram Bot

### Возможности
- Принимает **текстовые сообщения** на русском, узбекском и английском
- Автоматически распознаёт интент: доход, расход, отчёт, отмена
- Записывает: сумму, тип, категорию, дату, заметку
- Уточняет через **inline кнопки** если категория не определена
- Поддерживает **18 категорий** (6 доходов + 12 расходов)
- Работает для **всей команды** — общие данные, каждый видит общий отчёт

### Команды
| Команда | Описание |
|---------|----------|
| `/start` | Начать работу, выбор языка |
| `/report` | Отчёт за текущий месяц |
| `/undo` | Отменить последнюю запись |
| `/budget` | Бюджетные лимиты |
| `/reset` | Сбросить данные |
| `/language` | Сменить язык |
| `/help` | Помощь с примерами |

### Примеры фраз
```
"Получили 52 млн от продаж сегодня"   → income, Продажи
"Аренда 8.5 млн"                       → expense, Аренда
"Зарплата 18 миллионов вчера"          → expense, Зарплата
"52 mln savdodan tushdi"               → income, Savdo (узбекский)
"Сколько потратили за месяц?"          → отчёт за месяц
"отмени"                               → отмена последней записи
"Лимит аренда 10 млн"                  → установить бюджет
```

### Языки
Бот поддерживает **русский** и **узбекский** языки. При первом запуске предлагает выбрать язык. Сменить можно через `/language`.

## 📊 Part 2 — Web Dashboard

### Страницы
| Страница | Функционал |
|----------|-----------|
| **Обзор** | KPI карточки, график за 6 месяцев, последние записи, quick-add форма, онбординг для новых пользователей |
| **Транзакции** | Полный список, фильтры по типу и категории, поиск, inline редактирование и удаление |
| **Аналитика** | Donut charts по категориям, сравнительный bar chart за 6 месяцев |
| **Категории** | Все 18 категорий с иконками и статистикой использования |

### Синхронизация
Каждая транзакция из Telegram бота **мгновенно появляется на дашборде** через Railway API. Данные всегда актуальны.

## 🎯 Бонус фичи

### 1. Бюджетные лимиты
Установите месячный лимит по любой категории расходов:
```
"Лимит зарплата 20 млн"
```
Бот автоматически предупреждает:
- ⚠️ при достижении **80%** лимита
- ❌ при **превышении** лимита

### 2. Ежедневный автоотчёт
Каждый день в **18:00 по Ташкенту** бот автоматически отправляет всем пользователям сводку дня:
```
📊 Итоги дня — 26 апреля

💰 Доходы: 52 533 000 сум
💸 Расходы: 8 000 000 сум
📈 Прибыль: 44 533 000 сум

🔺 Топ расход: Аренда — 8 000 000 сум

Последние записи:
  · Продажи +52 533 000 сум
```

## 🚀 Запуск локально

### Bot
```bash
cd bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Создай .env файл
cp .env.example .env
# Заполни TELEGRAM_BOT_TOKEN

python main.py
```

### API
```bash
cd api
python3 -m venv venv
source venv/bin/activate
pip install fastapi uvicorn aiosqlite

uvicorn main:app --reload --port 8000
```

### Dashboard
```bash
cd dashboard
npm install
npm start
```

## 💡 Дизайн решения

- **Язык бота**: русский и узбекский — оба языка используются в малом бизнесе Узбекистана
- **Валюта**: UZS (узбекский сум) с форматированием `52 533 000 сум`
- **Категории**: подобраны под реальный малый бизнес Узбекистана — логистика, хлопок, сырьё, базар
- **Парсинг**: regex без внешних AI API — полностью бесплатно
- **Онбординг**: пустой дашборд показывает приветственный экран с инструкцией, не пустую страницу
